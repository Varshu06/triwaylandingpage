# Triway Technologies Landing Page

A polished, modern landing page showcasing Triway Technologies' partnership with Zoho, designed for UAE-based businesses seeking digital transformation solutions.

## Features

- **Hero Section**: Eye-catching hero with animated background, partner badge, and compelling CTAs
- **Value Proposition**: Four key benefits with animated cards
- **Solutions Overview**: Comprehensive Zoho solutions with detailed features
- **Why Choose Triway**: Trust signals, testimonials carousel, and competitive benefits
- **Call to Action**: Prominent CTAs with contact form
- **Resources & Footer**: Links to guides, case studies, and company information

## Design Highlights

- **Modern & Professional**: Clean design with professional color palette
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Smooth Animations**: Scroll-triggered animations, hover effects, and transitions
- **Interactive Elements**: Testimonial carousel, animated counters, form handling
- **Performance Optimized**: Lazy loading, efficient animations, and optimized assets

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with CSS Grid, Flexbox, and animations
- **JavaScript (ES6+)**: Interactive features and animations
- **Font Awesome**: Icons
- **Google Fonts**: Inter font family

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/Varshu06/triwaylandingpage.git
cd triwaylandingpage
```

2. Open `index.html` in your web browser or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js (http-server)
npx http-server

# Using PHP
php -S localhost:8000
```

3. Navigate to `http://localhost:8000` in your browser

## Customization

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #0066cc;
    --secondary-color: #00a8e8;
    --accent-color: #ff6b35;
    /* ... */
}
```

### Content
- Update text content directly in `index.html`
- Modify testimonials in the testimonials section
- Update statistics in the hero section

### Form Submission
Currently, the form shows a success notification. To connect to a backend:
1. Update the form submission handler in `script.js`
2. Replace the simulated API call with your actual endpoint
